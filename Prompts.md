Prompts Used - Week 1 Project (Prodesk IT Landing Page)

1. Initial Project Setup
I am building a professional landing page using only HTML, CSS, and JavaScript (no frameworks). Give me a clean structure for a responsive digital agency website.

2. Navbar Design
Create a responsive navbar with logo on the left and navigation links on the right. On mobile, convert it into a hamburger menu.

3. Hero Section
Design a modern hero section with a bold headline, subheading, and a call-to-action button for a digital agency website.

4. Services Section
Create a services section using Flexbox or Grid with three cards showing Web Development, SEO, and Marketing.

5. Glassmorphism UI
How can I implement glassmorphism effect using CSS (backdrop-filter, transparency) for cards and navbar?

6. Dark Mode
Implement a dark mode toggle using JavaScript that switches the entire theme of the website smoothly.

7. Mobile Responsiveness
Make the website fully responsive and ensure the navbar works properly on mobile devices using a hamburger menu.

8. Performance Optimization
My Lighthouse performance score is low. Optimize the project by removing render-blocking resources and improving load speed without using external CSS frameworks.

9. Lighthouse Fixes
How can I improve Accessibility and SEO scores in Lighthouse to reach close to 100?

10. Final Improvements
Convert the UI into a premium, modern landing page with smooth animations, proper spacing, and professional design suitable for a real company.

11. Deployment
Guide me step-by-step to deploy my static website using Vercel and generate a live link.
