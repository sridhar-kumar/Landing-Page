🌐 Prodesk IT - Digital Agency Landing Page

📌 Project Overview
This project is a professional landing page built for a fictional digital marketing agency called Prodesk IT.

The goal of this project is to demonstrate strong fundamentals in:
- HTML
- CSS (Flexbox & Responsive Design)
- Modern UI Design

🚀 Live Demo

🔗 https://landing-page-sable-ten-66.vercel.app


🎯 Features
- Fully Responsive Design (Mobile + Desktop)
- Modern UI with Gradient & Glassmorphism Effects
- Dark Mode Toggle 🌙
- Sticky Navbar
- Smooth Hover Animations
- Clean and Structured Code
- High Lighthouse Score (Performance 90+)

 🛠 Tech Stack
- HTML5
- CSS3 (Flexbox, Grid)
- Minimal JavaScript (for Dark Mode & Mobile Menu)
